Formulário vulnerável

Neste teste, uma página publicada em www.jstest.me contém um formulário de login e, ao mesmo tempo, um widget de previsão do tempo controlado por um script oriundo de um domínio diferente - <%= cdnDomain %>. Ocorre que este último domínio foi invadido por hackers: seus scripts foram modificados para que, além de apresentarem o clima, roubem as informações contidas no formulário e as enviem, silenciosamente, para um serviço no domínio invadido.

Como avaliar este teste: ao clique do botão "Entrar", o script malicioso fará uma requisição AJAX para o domínio <%= cdnDomain %>, perceptível somente pelo console de desenvolvimento do navegador (F12).
