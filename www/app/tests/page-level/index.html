Casos de teste em contexto de página

  • Formulário vulnerável, em que um script malicioso captura o conteúdo de um formulário de login/senha.
  • Formulário em shadow DOM, em que o conteúdo de um formulário passa a ser invisível a um script malicioso através de uma implementação ingênua do método proposto pela dissertação.
  • Interceptação de shadow DOM, em que o método ingênuo de ocultação do conteúdo de elemento HTML é invalidado por interceptação da API de shadow DOM.
  • Detecção de interceptação, em que é explorado um método para detectar a interceptação de APIs do navegador.
  • Formulário seguro em shadow DOM, em que o método para detectar APIs interceptadas é integrado ao método de ocultação do formulário HTML.
